---
title: نگهدارنده عکس
---

این کامپوننت یکی از پرمصرف ترین کامپوننت های ترب هست.

این کامپوننت شامل قابلیت های زیر می باشد:

* نمایش لودینگ اسکلتی قبل از نمایش تصویر
* نمایش placeholder با سایز مناسب در حالتی که تصویر خواسته شده لود نشود و به مشکل بخورد
* پذیرش استایل های بیرونی

```
import { ImageHolder } from '@torob/react-common'

<ImageHolder
  src={'http://example.com/cute_cat.jpg'}
  width="200"
  height="200"
/>
```
