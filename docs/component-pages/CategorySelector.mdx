---
title: انتخاب کننده دسته بندی
---

این کامپوننت در پنل های مختلف برای انتخاب دسته بندی استفاده می شود. این کامپوننت در دو حالت دینامیک و استاتیک قابل استفاده است. 

کامپوننت استاتیک، بعنوان prop، لیست دسته بندی هایی که باید نمایش داده شوند را دریافت میکند


```
const catsList = [{
  title: 'دسته بندی اول',
  id: 'cat1',
},
{
  title: 'دسته بندی دوم',
  id: 'cat2',
},
{
  title: 'دسته بندی سوم',
  id: 'cat3',
}]
<StaticCategorySelector categories={catsList} />
```
