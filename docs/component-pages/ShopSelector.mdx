---
title: انتخاب کننده فروشگاه
---

این کامپوننت در پنل های مختلف برای انتخاب فروشگاه فعال استفاده می شود. این کامپوننت در دو حالت دینامیک و استاتیک قابل استفاده است. 

کامپوننت استاتیک، بعنوان prop، لیست فروشگاه هایی که باید نمایش داده شوند را دریافت میکند

<StaticShopSelector shops={[{
  name: 'فروشگاه اول',
  id: 'shop1',
},
{
  name: 'فروشگاه دوم',
  id: 'shop2',
},
{
  name: 'فروشگاه سوم',
  id: 'shop3',
}]} />

```
const shopsList = [{
  name: 'فروشگاه اول',
  id: 'shop1',
},
{
  name: 'فروشگاه دوم',
  id: 'shop2',
},
{
  name: 'فروشگاه سوم',
  id: 'shop3',
}]
<StaticShopSelector shops={shopsList} />
```
